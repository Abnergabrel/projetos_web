O elemento table (Tabela)

O eletemnto table (Tabela) representa dados tabulares ou seja, informações que são representados em linhas e colunas.

exemplo

Tabela simples com cabeçalho

Primeiro nome
Sobrenome
Whashington Paiva
Luis Paiva

Elementos de tabelas

  • table - tabelas;
  • th (table header) - cabeçalho de coluna;
  • tr (table row) - linha da tabela;
  • td (table data) - valor/conteúdo de célula.
